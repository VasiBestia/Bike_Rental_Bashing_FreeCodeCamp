## ðŸ—„ Database Schema

The database includes the following main tables:

- **bikes**
  - `bike_id` (PRIMARY KEY)
  - `type`
  - `size`
  - `available` (BOOLEAN)

- **customers**
  - `customer_id` (PRIMARY KEY)
  - `name`
  - `phone` (UNIQUE)

- **rentals**
  - `rental_id` (PRIMARY KEY)
  - `customer_id` (FOREIGN KEY)
  - `bike_id` (FOREIGN KEY)
  - `date_rented`
  - `date_returned`
